# Overview of MOD550 Task 3 Files

## Dataset
* Processed dataset used for all model training and evaluation: **F1_data.csv**
* Unseen 2025 race data used for testing: **F1_data_2025.csv**

## Class Modules
* Handles downloading, cleaning, and preparing the raw F1 data: **data_acquisition.py**
* Main analysis class (trains models, evaluates performance, plots results): **f1_analytics.py**

## Notebook
* The main notebook for data loading, modelling, testing, and plotting: **task3_notebook.ipynb**

## Report and Presentation
* Final written report summarizing approach, methods, and results: **MOD550_Project_Final_Report.pdf**
* Slides used for presenting the project in class: **MOD550_Project_Presentation.pdf**
